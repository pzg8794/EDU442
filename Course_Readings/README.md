## About This Reading List

These readings cover:
- **Critical race theory** and education
- **Intersectionality** (race, class, gender, disability, sexuality)
- **Pedagogy** for justice, belonging, and cultural sustenance
- **Structural analysis** (neoliberalism, school-to-prison pipeline, precarious knowledge)
- **Identity-conscious practice** and decolonization

Each week's readings are designed to build understanding progressively. See the main [EDU442-Notes.md](../Notes/EDU442-Notes.md#4-readings-schedule--quick-study-summaries) for:
- Detailed summaries of each reading
- Core claims and key arguments
- EQUITAS-focused reflection prompts
- Evidence citations and "what to skim" guidance

## Study Strategy

1. **Start with the summary** in EDU442-Notes.md to get the core claim
2. **Skim the introduction** of the actual PDF to confirm focus
3. **Read actively** for evidence that connects to your own teaching/research questions
4. **Annotate** with questions and connections
5. **Synthesize** using the weekly workflow template (synthesis, 3 claims, EQUITAS lens, 2 concrete moves)

All summaries in the notes are **study accelerators**, not replacements for reading. They help you navigate quickly and prepare discussion/writing.
