# Slides

This directory is for presentation slides (PowerPoint, Beamer, Keynote, etc.) created for class presentations and group work.

**Current status:** Empty (will be populated as presentations are created)

## When to add files here:
- Week 8 Presentation slides
- Impacting Practice final presentation materials
- Any group presentation decks

## Naming convention:
Use clear names like:
- `EDU442_Week8_Presentation_GROUP_NAME.pptx`
- `EDU442_ImpactingPractice_Presentation.pdf`
- `EDU442_[TOPIC]_[DATE].key` (for Keynote)
