# Class Activities

This directory contains in-class activities, handouts, and collaborative materials used during class meetings.

## Files

**Location:** This directory

- [How to write productive questions.gdoc](How%20to%20write%20productive%20questions.gdoc) - Guide on crafting high-quality discussion questions for group work and facilitation
- [Matrix of Domination Examples.gdoc](Matrix%20of%20Domination%20Examples.gdoc) - Examples and framework for analyzing systems of domination (Collins, hooks)

## Materials by Use

**For Discussant Activities:**
Use [How to write productive questions.gdoc](How%20to%20write%20productive%20questions.gdoc) as reference when crafting your discussion questions for each class session.

**For Intersectionality Mapping:**
Use [Matrix of Domination Examples.gdoc](Matrix%20of%20Domination%20Examples.gdoc) to understand how multiple systems of domination intersect and shape lived experience.
